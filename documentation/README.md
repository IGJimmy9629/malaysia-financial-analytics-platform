# Project Documentation

This folder contains the presentation materials for the **Malaysia Financial Analytics Platform**.

The presentation focuses on the visual analysis of Malaysia's economic and financial performance between **2000 and 2024**. It summarises the findings generated from the analytical datasets and presents them through static, animated and interactive visualisations.

The technical implementation of the project, including the Apache Hive ETL pipeline, Python validation workflow, R visual analytics scripts and repository structure, is documented separately in the main repository `README.md`.

# Presentation Overview

The presentation covers the following topics:

1. Malaysia's economic performance (2000–2024)
2. GDP growth trends
3. Inflation performance
4. Unemployment trends
5. Household income growth
6. Household financial behaviour
7. Borrowing and saving environment
8. Domestic credit
9. Banking stability
10. Economic relationships
11. Executive economic assessment
12. Interactive dashboard overview

---

# Purpose

The presentation is intended to communicate the analytical findings through visual storytelling.

It complements the technical implementation contained in the main repository by focusing on the interpretation of Malaysia's economic indicators rather than the underlying data engineering process.

For details of the implementation, workflow and project architecture, please refer to the main repository `README.md`.
